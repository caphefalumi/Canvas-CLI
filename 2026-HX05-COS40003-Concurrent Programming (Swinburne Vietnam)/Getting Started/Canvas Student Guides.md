# Canvas Student Guides

Canvas is the Learning Management System (LMS) used for all units in the Faculty of Science, Engineering and Technology (FSET). For an introduction to the Canvas interface [https://swinburne.instructure.com/courses/75936/assignments/syllabus] and functionality, please view the video below:

You can take a look at this Canvas Overview for Students [https://vimeo.com/74677642].

Instructions explaining how to login to Canvas for the first time can be found in the document Student Login to Canvas [https://swinburne.instructure.com/courses/75936/files/43188282/download?verifier=6d983PsYnMlUB0MW5i8Ofhz5qmjx0p0aEPE9yvx9&wrap=1].

Students can access the Canvas Student Guides [https://community.canvaslms.com/docs/DOC-10701] and Frequently Asked Questions for Swinburne Students [https://www.swinburne.edu.au/current-students/get-started/access-systems/access-student-systems/canvas/], or can Report a Problem to Canvas [https://swinburne.instructure.com/courses/75936/pages/canvas-user-guides] from the Help Menu. 

You can also navigate Canvas directly from your phone using the Canvas Student App. [https://community.canvaslms.com/docs/DOC-14241-67952843574]
